## Password Reset Procedure

### Objective

This SOP outlines the steps to reset a user's password in the directory system.

### Key Steps

 

**Step 1: Access the Directory** [0:00](https://loom.com/share/0d8969eade404b6eaff6c8060efddff3?t=0)

![Active Directory](https://github.com/user-attachments/assets/c769ee04-b03e-4f7f-aee5-83c556a914fe)


- Locate the search function on your computer.
- Click on the directory option.

 

**Step 2: Find the User** [0:11](https://loom.com/share/0d8969eade404b6eaff6c8060efddff3?t=11)


- In the directory, search for the user whose password needs to be reset.
- For example, look for 'John Doe'.

 

**Step 3: Reset the Password** [0:34](https://loom.com/share/0d8969eade404b6eaff6c8060efddff3?t=34)

![Reset Password](https://github.com/user-attachments/assets/86882834-258a-489d-92bc-5ef5ca4f91ca)


- Once you have located the user, proceed to reset the password.
- Enter the new password in the designated field.

 

**Step 4: Confirm the Change** [0:34](https://loom.com/share/0d8969eade404b6eaff6c8060efddff3?t=34)

![confirm change](https://github.com/user-attachments/assets/a6e155b8-c37d-48b8-8898-68a3b2228e65)

- Ensure that the password has been successfully changed.
- Verify that the system reflects the new password.
